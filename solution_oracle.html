Features
Unbreakable Enterprise Kernel
  • Developed and tested to support demanding workloads, such as Oracle Database.
  • Tracks Linux mainline closely.
  • Allows features such as Btrfs, DTrace, OCFS2, and Linux Containers to be used across multiple major releases (Oracle Linux 6. 4 and later).
  • Able to parallelize network and disk IO.
  • Efficiently runs on systems with many cores and threads and NUMA nodes.
Docker
  • Optimized framework for deploying single-purpose, isolated Linux Containers.
  • Supports full Linux Container lifecycle: build, ship, distribute, and deployment of applications.
  • Leverages modern kernel technologies including cgroups, SElinux, and copy-on-write filesystems.
  • Oracle Linux images available in the official Docker Hub.
  • Oracle WebLogic Server is certified to run on Docker in Oracle Linux.
Ksplice
  • Enables zero-downtime kernel updates for critical security updates and bug fixes.
  • Minimizes security risks by easily keeping system up to date without downtime.
  • Improves application availability for users.
  • Provides kernel diagnostics on production systems.
  • Easily integrates into your environment’s change management practices with rollback feature.
  • Support for Ksplice is included with Oracle Linux Premier Support subscriptions at no additional charge.
DTrace
  • Provides a single view of the software stack.
  • Traces kernel, system libraries, and applications.
  • Quickly identify performance bottlenecks on production systems by defining probe points live, at runtime.
  • Enables static tracing using instrumentation compiled into kernel and applications.
  • Support for DTrace is included with Oracle Linux Support at the Basic and Premier levels at no extra charge.
Secure, Enterprise-Class Support
  • High-quality, enterprise-class, secure Oracle Linux support
  • Oracle Linux Premier Support includes zero-downtime kernel updates with Ksplice
  • Management and clustering software is included at no additional charge
  • Oracle Premier Support for Systems includes Oracle Linux support (no need to repurchase)
  • Oracle Cloud subscription includes Oracle Linux Premier Support at no additional cost
  • Includes premier backports, comprehensive legal indemnification, and full-stack testing
